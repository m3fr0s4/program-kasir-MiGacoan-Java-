# 🍜 Sistem Kasir Restoran Mi Gacoan

Mini Project UAS Algoritma Pemrograman Lanjut

## Deskripsi

Program kasir sederhana berbasis Java menggunakan konsep Object-Oriented Programming (OOP).

## Fitur

- Input data pelanggan
- Status Member (diskon 5%)
- Menu makanan dan minuman
- Kustom level pedas Mie Gacoan & Mie Hompimpa
- Perhitungan subtotal
- Perhitungan diskon
- Pembayaran Cash / QRIS
- Cetak struk

## Konsep OOP yang Digunakan

- Class & Object
- Inheritance
- Constructor
- Constructor Overloading
- Method Overloading
- Encapsulation
- Method Overriding

## Struktur Project

```
MiGacoanOOP
│
├── Main.java
├── Menu.java
├── Makanan.java
├── Minuman.java
├── Pelanggan.java
├── Pesanan.java
└── Transaksi.java
```

## Bahasa Pemrograman

- Java (JDK 25)
